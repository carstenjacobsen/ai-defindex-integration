# AI DeFindex Integration

DeFindex is a decentralized protocol that makes yield simple and accessible. DeFindex empowers wallet providers, developers, and crypto users to integrate and access a wide range of strategies—bridging DeFi complexity into a plug-and-play solution for yield generation.

There are two main ways to integrate DeFindex into your project: using the DeFindex SDK to build a custom integration, or using the DeFindex Vaults to access pre-built strategies. The SDK is a TypeScript library that provides a simple interface for interacting with the DeFindex protocol, while the Vaults are ready-to-use smart contracts that you can interact with directly from your frontend or backend.

## Why Use DeFindex?
- Plug-and-play yield → no need for deep DeFi expertise
- Composable strategies → support multiple assets and strategies per asset
- Secure architecture → audited, with built-in safeguards like rebalance and rescue functions
- Aligned incentives → wallets earn whenever their users earn
- Future-proof → easily extend strategies, integrate real-world assets, and unlock new income streams

Learn more about DeFindex and get started with these resources:

- [Documentation](https://docs.defindex.io)
- [DeFindex Website](https://www.defindex.io)
- [DeFindex SDK](https://github.com/paltalabs/defindex-sdk)
- [DeFindex Vaults](https://www.defindex.io/vaults)

## Vibe Coding
DeFindex is a comprehensive protocol with many features and capabilities, but thanks to an easy-to-use API, great documentation and a lot of example projects and open source projects, Claude Code is very capable of integrating DeFindex.

A simple prompt to build a simple DeFindex-based dApp could look like this:

```
create an application using stellar freighter wallet and DeFindex to make my own yield earning application.
```

This will build a Next.js dapp with the Freighter wallet integration and the DeFindex integration, allowing users to connect their wallet, create vaults, deposit XLM, and earn yield on their assets.

### Potential Issues
Claude Code seems to be able to generate code for the DeFindex integration by itself, only blockers seem to be:

- **API key** - requests to the API will fail without an API key, and error messages from the dapp may not be clear about this. The API key must be obtained from the DeFindex website, and can't be generated by Claude Code. Create API key by [registering here](https://api.defindex.io/register).
- **Create vault** - the user must have created a vault, and selected yield strategies, before the code will work. Create a vault [here](https://app.defindex.io).
- **XLM transactions** - The dapp will depend on the user being able to send XLM transactions from their wallet, otherwise deposits to the vault will fail, and it may not be clear if the problem is with the wallet or the DeFindex integration.
